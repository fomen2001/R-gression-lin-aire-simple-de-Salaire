# R-gression-lin-aire-simple-de-Salaire
# Projet  : Régression Linéaire

 Régression Linéaire Simple (Prédiction de Salaire)

Cette section utilise la méthode des **Moindres Carrés Ordinaires (MCO)** pour ajuster un modèle mathématique à un ensemble de données.

### Objectifs Techniques

* Minimiser la somme des carrés des écarts entre les données réelles (Y) et les prédictions du modèle.


* Prédire le salaire en fonction des années d'expérience (X).



### Stack Technique

* 
**Langage :** Python.


* 
**Bibliothèques :** NumPy, Pandas, Matplotlib et Scikit-learn.



### Méthodologie

1. 
**Chargement :** Utilisation du fichier `Salary_Data.csv`.


2. 
**Division :** Répartition des données en ensemble d'apprentissage ($2/3$) et ensemble de test ($1/3$) avec `train_test_split`.


3. 
**Entraînement :** Utilisation de la classe `LinearRegression` de scikit-learn.


4. 
**Visualisation :** Affichage d'un nuage de points (rouge) et de la ligne de prédiction (bleu) via Matplotlib.
